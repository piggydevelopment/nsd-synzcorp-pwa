# แผนการแก้ไขปัญหา PWA จอดำ (Plan)

## 1. การแก้ไขใน `public/manifest.json`
- **แก้ `start_url`:** เปลี่ยนจาก `"start_url": "https://corp.synzofficial.com/login"`
  ให้เป็น **`"start_url": "."`** หรือ **`"start_url": "/login"`** (Relative Path) เพื่อให้ PWA ใช้งานได้โดยไม่ต้องผูกติดกับโดเมนใดโดเมนหนึ่งแบบตายตัว และทำงานร่วมกับ Service Worker Cache ได้แม่นยำขึ้น
- **เพิ่ม `"scope": "/"`** (ถ้าจำเป็น) เพื่อกำกับขอบเขตการทำงานของ PWA

## 2. การจัดการ Service Worker และ Cache
- ตรวจสอบโฟลว์ของ `serviceWorkerRegistration.js`
- พิจารณาเพิ่มโค้ดป้องกันกรณี JS ทำงานผิดพลาดหรือเกิดข้อขัดข้องระหว่างโหลดหน้าจอ ให้แสดง Fallback UI 

## 3. การตรวจสอบ Routing หน้า Login
- ดูความเสถียรของ `<BrowserRouter>` ว่ารับกับ Path ของ PWA ได้ถูกต้อง

**Note:** ตามนโยบายข้อ 10 และ 11 (ใช้ .md วางแผนและให้ตรวจสอบก่อนเริ่มพิมพ์แก้ไขระบบ) ผมจึงเตรียมสรุปแผนตรงนี้ไว้ให้คุณดูก่อนครับ
